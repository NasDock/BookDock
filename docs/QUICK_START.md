# 快速入门

本文档帮助你在几分钟内完成 BookDock 的初始配置并开始阅读。

---

## 📋 前提条件

- Node.js >= 18
- pnpm >= 8（推荐 9.x）
- 如果使用后端服务：PostgreSQL + Redis（或使用 Docker 一键启动）

---

## 1. 安装 & 启动

### 方式一：Docker 完整部署（推荐）

```bash
git clone https://github.com/mmdctjj/BookDock.git
cd BookDock
cp .env.example .env
# 编辑 .env 配置 NAS 路径和数据库密码
docker-compose up -d
open http://localhost
```

### 方式二：本地开发

```bash
# 安装依赖
pnpm install

# 启动 API Server（终端 1）
cd apps/server
pnpm dev

# 启动 Web 前端（终端 2）
cd apps/web
pnpm dev

# 访问 http://localhost:5173
```

---

## 2. 登录 / 注册

### 创建账户

1. 打开 BookDock，点击 **「注册」**
2. 输入手机号，获取验证码
3. 设置密码，完成注册
4. 自动登录进入书架

### 登录已有账户

1. 点击 **「登录」**
2. 选择 **手机号 + 验证码** 或 **手机号 + 密码** 登录
3. 登录成功后自动跳转书架

### 离线 / NAS 模式

如果没有网络连接，可以直接连接 NAS 的本地账户使用，无需注册。

---

## 3. 添加第一本书

### 方法一：上传电子书

1. 进入 **书架** 页面
2. 点击右上角 **「+ 添加书籍」**
3. 选择本地文件（支持 EPUB / PDF / MOBI / TXT）
4. 等待上传完成，书籍出现在书架上

### 方法二：从书源同步

1. 进入 **「设置」→「书源管理」**
2. 点击 **「添加书源」**
3. 选择类型（WebDAV / SMB / FTP）
4. 填写 NAS 地址、用户名、密码
5. 保存后点击 **「同步」**，书籍自动导入书架

### 方法三：管理后台上传（管理员）

1. 进入 **「管理后台」→「书籍管理」**
2. 点击 **「上传书籍」**
3. 选择文件或指定 NAS 路径
4. 上传完成后，所有用户可在书架看到

---

## 4. 开始阅读

1. 在书架中找到目标书籍，点击封面或标题
2. 阅读器加载电子书内容
3. 使用以下方式翻页：

   | 操作 | 方式 |
   |------|------|
   | 翻页 | 点击屏幕左/右侧，或使用 ← → 方向键 |
   | 滑动 | 在屏幕上左右滑动 |
   | 目录 | 点击屏幕中央唤出目录面板 |
   | 书签 | 点击右上角书签图标添加书签 |
   | 设置 | 点击右上角「Aa」调整字体/主题 |

### 阅读器设置

点击屏幕中央 → 点击右下角 **「Aa」**：

- **字体大小**：滑动调节
- **主题**：浅色 / 深色 / 护眼 / Sepia
- **行间距**：紧凑 / 适中 / 宽松
- **字体**：系统字体 / 黑体 / 宋体

---

## 5. 开启听书模式

BookDock 支持两种 TTS 模式：

### 模式一：浏览器 TTS（无需配置，立即可用）

1. 在阅读器中点击右上角 **「🔊 听书」** 按钮
2. 自动使用浏览器内置语音合成
3. 可调节语速（0.5x ~ 2.0x）

### 模式二：服务器 TTS（高质量，需配置）

1. 在 **「设置」→「TTS 设置」** 中填入 TTS 服务地址
2. 返回阅读器，点击 **「🔊 听书」**
3. 可选择不同语音（男声 / 女声 / 中文 / 英文等）
4. 支持段落跳转、语速 / 音量精细调节

### 听书控制

```
[◀◀] [▶️/⏸] [▶▶]   ← 上一句 / 播放-暂停 / 下一句
语速: [━━━●━━] 1.0x  ← 拖动调节
音量: [━━━●━━] 80%   ← 拖动调节
```

阅读过程中，TTS 会跟随当前阅读位置自动朗读。长按文字可选中段落进行单独朗读。
