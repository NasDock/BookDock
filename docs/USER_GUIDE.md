# 用户指南

本文档详细介绍 BookDock 各功能模块的使用方法。

---

## 📚 书架管理

### 切换视图

书架支持两种视图切换：
- **网格视图**：书籍封面大图展示，适合浏览封面
- **列表视图**：每行显示书名、作者、阅读进度，适合快速扫描

点击右上角视图切换按钮即可切换。

### 添加书籍

#### 本地上传
1. 点击 **「+ 添加书籍」**
2. 选择文件（支持 EPUB / PDF / MOBI / TXT）
3. 等待上传完成

#### 从书源同步
1. 前往 **「设置」→「书源管理」** 添加书源
2. 点击书源的 **「同步」** 按钮
3. 同步完成后自动出现在书架

#### NAS 直连
1. 在书源中添加 NAS 连接（WebDAV / SMB / FTP）
2. 配置 NAS 路径指向电子书目录
3. 启用自动同步，书籍自动发现并导入

### 删除书籍

1. 在书架中长按书籍（或右键）
2. 选择 **「删除」**
3. 确认删除（仅从书架移除，不删除源文件）

### 搜索书籍

- **快速搜索**：在书架顶部搜索框输入书名 / 作者名
- **高级筛选**：点击筛选按钮，支持按格式、阅读状态、添加日期筛选

### 书籍详情

点击书籍封面进入详情页，可查看：
- 书籍基本信息（书名、作者、格式、文件大小）
- 阅读进度百分比和上次阅读时间
- 目录结构
- 书签和高亮列表
- 相关推荐

## 📖 阅读器使用

### 翻页操作

| 平台 | 操作 |
|------|------|
| Web / PWA | 点击左/右侧翻页，键盘 ← → 翻页 |
| Desktop | 点击翻页，键盘 ← →，或鼠标手势 |
| Mobile | 点击翻页，左右滑动翻页 |

### 字号调节

1. 点击屏幕中央唤出控制栏
2. 点击右下角 **「Aa」**
3. 拖动字号滑块（12px ~ 32px）

### 主题模式

支持四种主题：

| 主题 | 说明 |
|------|------|
| 浅色 | 白底黑字，适合光线充足环境 |
| 深色 | 深灰底白字，适合夜间阅读 |
| 护眼 | 米黄底棕字，降低蓝光刺激 |
| Sepia | 经典羊皮纸风格 |

### 目录导航

1. 点击屏幕中央唤出控制栏
2. 点击 **「目录」** 按钮
3. 在目录面板中点击章节名跳转

### 书签功能

**添加书签**：
- 点击阅读器右上角 **书签图标**，添加当前页为书签
- 或选中文字后，选择 **「添加书签」**

**管理书签**：
- 进入 **「我的书签」** 页面
- 可查看所有书签，按书籍分组
- 点击书签直接跳转到对应位置

### 高亮 & 笔记

**添加高亮**：
1. 长按选中文字
2. 选择高亮颜色（黄 / 绿 / 蓝 / 红）
3. 高亮自动保存到云端

**导出笔记**：
1. 进入书籍详情页的 **「笔记」** 标签
2. 可查看所有高亮和笔记
3. 点击 **「导出」** 生成 Markdown 文件

## 🔊 TTS 听书

### 播放控制

```
[上一句] [播放/暂停] [下一句]  ← 基础控制
语速:  ●━━━━━━━━━  1.0x      ← 0.5x ~ 2.0x
音量:  ●━━━━━━━●  80%       ← 0% ~ 100%
```

- **上一句 / 下一句**：按段落跳转
- **播放 / 暂停**：空格键快捷键（Web 端）
- **语速**：中文推荐 1.0x ~ 1.25x，英文推荐 1.0x

### 语速调节

| 语速档位 | 适用场景 |
|---------|---------|
| 0.5x | 慢速学习 / 复杂内容 |
| 0.75x | 初次尝试听书 |
| 1.0x | 正常语速（推荐） |
| 1.25x | 较快阅读 |
| 1.5x ~ 2.0x | 快速回顾 |

### 语音选择

在 **「设置」→「TTS 设置」** 中选择语音：

- **Web Speech API**：使用浏览器内置语音（离线可用，音质一般）
  - 中文：`zh-CN` 系列语音
  - 英文：`en-US` 系列语音

- **服务器 TTS**：高质量云端合成（需配置 TTS 服务）
  - 支持多种音色（男声/女声/童声）
  - 支持中英混合朗读
  - 支持情感语调调节

### 后台播放

开启听书模式后，切换到其他页面或锁屏，TTS 朗读不中断（移动端 PWA 支持）。

## ⚙️ 设置说明

### 账户设置

- **修改密码**：设置 → 账户安全 → 修改密码
- **绑定手机**：设置 → 账户安全 → 更换手机号
- **注销账户**：设置 → 账户安全 → 注销账户（不可逆）

### 阅读设置

- **默认主题**：选择打开阅读器时的默认主题
- **默认字号**：选择打开阅读器时的默认字号
- **翻页动画**：开启/关闭翻页过渡动画
- **双击阅读**：开启后双击屏幕特定区域可快捷翻页

### TTS 设置

- **默认语音引擎**：选择 Web Speech / 服务器 TTS
- **默认语速**：设置听书默认语速
- **默认音量**：设置听书默认音量
- **TTS 服务地址**：配置服务器 TTS 的 URL（`http://your-tts-server:5000`）

### 书源管理

- **添加书源**：WebDAV / SMB / FTP 配置
- **编辑书源**：修改连接信息
- **删除书源**：移除后不同步该书源
- **定时同步**：设置自动同步周期（关闭/每小时/每天）

### 通知设置

- **阅读提醒**：设置每日阅读目标提醒
- **同步通知**：书源同步完成时通知
- **会员到期提醒**：会员到期前 7 天通知

### 清理缓存

- **清除缓存**：清理本地已下载的书籍缓存（不影响云端数据）
- **清除阅读历史**：重置所有阅读进度记录
